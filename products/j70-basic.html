Spruce C24 insert · Economy grade

Key Features

  • Identical fixings: Drop-in interchangeable with other J70 variants — same slots, same end caps, same plywood nailing workflow
  • Insert tuned for application: European Spruce C24
  • Tested ultimate capacity: Structural testing conducted at IIT Madras + Bureau Veritas
  • Hot-dip galvanised steel tube: 70-micron post-slot HDG for corrosion resistance on wet sites
  • Nail through slots: Plywood fixes directly via through-slots — no proprietary accessories
  • Factory-sealed end caps: Protect the timber insert from moisture and site impact
