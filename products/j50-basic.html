J50 StiMBERTM Basic

Medium-duty secondary beam · Economy

Best for: Secondary beam applications — cost-focused projects. The steel tube, end caps, outer dimensions and fixing geometry are identical to other J50 variants — only the internal timber insert differs.

Key Features

  • Identical fixings: Drop-in interchangeable with other J50 variants — same slots, same end caps, same plywood nailing workflow
  • Insert tuned for application: European Spruce C24
  • Tested ultimate capacity: Structural testing conducted at IIT Madras
  • Hot-dip galvanised steel tube: 70-micron post-slot HDG for corrosion resistance on wet sites
  • Nail through slots: Plywood fixes directly via through-slots — no proprietary accessories
  • Factory-sealed end caps: Protect the timber insert from moisture and site impact

Before You Use

All installation, striking, storage and handling must comply with the StiMBERTM User Manual & Standard Operating Procedures. Every operator and supervisor on site must be briefed on this document before commencing work.
